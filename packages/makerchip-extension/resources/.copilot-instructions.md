# Makerchip Data Workspace

**This workspace folder is for reference data only** - managed by the Makerchip VS Code extension.

## ⚠️ Do NOT create user files here

- User TL-Verilog projects → Create in your own workspace folders
- Temporary experiments → Use `tmp/` subdirectory only
- Files here may be overwritten during extension updates

## What's Here

- `resources/` - TL-Verilog specs, docs, examples, libraries (read-only reference)
- `compile-cache/` - Recent compilation results for debugging
- `tmp/` - Temporary workspace (auto-cleaned during updates)
