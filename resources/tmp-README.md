# Temporary Files

Use this directory for temporary files and experiments.

## Automatic Cleanup

When you update the Makerchip workspace, this directory is **moved** to the system temp directory (`vscode-makerchip-<timestamp>/`), which is cleared on reboot.

## Usage

**Use for**: VIZ outputs, test files, experiments, debug artifacts

**Don't use for**: Production designs, files to keep long-term, version-controlled files

For organized projects, create files in your workspace folder instead.
